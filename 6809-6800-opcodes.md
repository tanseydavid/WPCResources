6809 - 6800 OpCodes
===================



| 6809 Instruction(s)              | 6800 equivalent                          |
|----------------------------------|------------------------------------------|
| PSHS B; ADDA ,S+                 | ABA                                      |
| PSHS B; CMPA ,S+                 | CBA                                      |
| ANDCC #$FE                       | CLC                                      |
| ANDCC #$EF                       | CLI                                      |
| ANDCC #$FD                       | CLV                                      |
| LEAS -1,S                        | DES                                      |
| LEAX -1,X                        | DEX                                      |
| LEAS 1,S                         | INS                                      |
| LEAX 1,X                         | INX                                      |
| PSHS A *                         | PSHA                                     |
| PSHS B *                         | PSHB                                     |
| PULS A *                         | PULA                                     |
| PULS B *                         | PULB                                     |
| PSHS B; SUBA ,S+                 | SBA                                      |
| ORCC #$01                        | SEC                                      |
| ORCC #$10                        | SEI                                      |
| ORCC #$02                        | SEV                                      |
| TFR A,B; TSTA                    | TAB                                      |
| TFR A,CC                         | TAP                                      |
| TFR B,A; TSTA                    | TBA                                      |
| TFR CC,A                         | TPA                                      |
| TFR S,X                          | TXS                                      |
| CWAI #$FF                        | WAI                                      |
| CWAI #$EF                        ; CLI; WAI                                 |
